### 介绍

>这个分支是AngelCrystal系列的设计标准及语言，为什么叫AGUI？因为没名字起了，为了让之后的UI统一设计语言，同时我也会发布基于xml的(Android)界面[main.xml]已发布这个UI设计的网页端以及移动端

另外可克隆本仓库做出自己的版本并发布！！！
